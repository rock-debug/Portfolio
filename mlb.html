PITCH COMPLETION | PYBASEBALL PIPELINE

Before a home run can be modeled, the data must be captured.

I built an automated Python data pipeline using pybaseball to extract every pitch of the 2025 MLB season. The dataset was cleaned, trimmed to remove anomalies, and structured for maximum indexing performance under Power BI limits.

The Contact.

STAR SCHEMA | POWER BI DATA MODEL

A flawless swing requires perfect alignment—the exact premise of a robust Star Schema.

I engineered a deeply relational model centered around a FactPitch table, logging exactly one row per physical pitch thrown.

The Apex.

ADVANCED DAX MEASURES
